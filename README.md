# nomade.la

Landing page for NOMAD'E Los Angeles — LA born handmade jewelry for the visionary individual.

## Features

- Image + video slideshow with crossfade transitions
- Mobile-optimized layout
- Modal-based contact interface (Instagram, email)
- SEO meta tags and OpenGraph support

## Technology Stack

- Vanilla JavaScript
- CSS3 (backdrop-filter, transitions, object-fit)
- Responsive Design
- GitHub Pages hosting

## Setup

1. Clone the repository
2. Add images to `images/slideshow/`
3. Update media paths in `script.js` if needed
4. Deploy via GitHub Pages

## Directory Structure

```
├── images/
│   ├── nomade-logo.png
│   └── slideshow/       # Product images for slideshow
├── index.html           # Main HTML file
├── script.js            # Image + video slideshow
├── manifest.json        # Web app manifest
├── robots.txt           # Search engine directives
├── sitemap.xml          # Site structure for search engines
└── CNAME                # Custom domain config
```
